# TO DO list

A simple flutter app using BLoC architecture pattern to save, view and remove TO DO list 

## Features

* Add a TO DO item
* View all TO DO items
* Remove an item by tapping remove button 

## Technology Used
- Dart (Primary Language)
- Architecture Pattern : **BLoC**
